Detailed guide for every Yable feature. Each section explains what the feature does, how to enable it, a code example, and relevant notes.

- [Sorting](/docs/features/sorting) — Sort rows by one or more columns with built-in and custom sort functions.
- [Filtering](/docs/features/filtering) — Filter rows with per-column filters and a global search.
- [Pagination](/docs/features/pagination) — Slice the row model into pages with configurable page sizes.
- [Cell Editing](/docs/features/cell-editing) — Edit cell values inline with validation, formatting, and multiple input types.
- [Column Pinning](/docs/features/column-pinning) — Freeze columns to the left or right edge of the table.
- [Column Resizing](/docs/features/column-resizing) — Drag column borders to resize columns.
- [Column Visibility](/docs/features/column-visibility) — Show or hide columns dynamically.
- [Column Ordering](/docs/features/column-ordering) — Reorder columns programmatically.
- [Column Grouping](/docs/features/column-grouping) — Group rows by column values.
- [Aggregation](/docs/features/aggregation) — Calculate summary values for grouped rows.
- [Row Selection](/docs/features/row-selection) — Select rows with single-click or checkboxes.
- [Row Expanding / Master-Detail](/docs/features/row-expanding) — Expand rows to show detail content below them.
- [Row Pinning](/docs/features/row-pinning) — Pin rows to the top or bottom of the table.
- [Tree Data](/docs/features/tree-data) — Display hierarchical/nested data with expand/collapse controls.
- [Pivot Tables](/docs/features/pivot-tables) — Transform flat data into cross-tabulated views.
- [Undo / Redo](/docs/features/undo-redo) — Track edit history and allow users to undo/redo changes.
- [Clipboard](/docs/features/clipboard) — Copy, cut, and paste table data with configurable delimiters.
- [Fill Handle](/docs/features/fill-handle) — Drag to auto-fill adjacent cells with pattern detection.
- [Formulas](/docs/features/formulas) — A spreadsheet-grade formula engine with built-in functions.
- [Async Commits](/docs/features/async-commits) — Save cell edits to a backend with optimistic updates, error handling, and conflict detection.
- [Export](/docs/features/export) — Export table data to CSV or JSON format.
- [Event System](/docs/features/event-system) — A typed event emitter for responding to table interactions.
- [i18n](/docs/features/i18n) — Internationalization support for table UI strings.
