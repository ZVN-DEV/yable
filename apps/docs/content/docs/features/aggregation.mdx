Calculate summary values for grouped rows. Aggregation functions run on each group's leaf rows.

<Callout type="info">
  Grouped/aggregated row rendering is turnkey. Set `initialState.grouping` (or call
  `table.setGrouping([...])`) and give a column an `aggregationFn` — `<Table>` renders
  collapsible group headers with the rolled-up aggregate in each column automatically. See
  [Column Grouping](./column-grouping) for the grouping setup.
</Callout>

### Built-in Aggregation Functions

| Name          | Description               |
| ------------- | ------------------------- |
| `sum`         | Sum of numeric values     |
| `min`         | Minimum numeric value     |
| `max`         | Maximum numeric value     |
| `extent`      | `[min, max]` tuple        |
| `mean`        | Average of numeric values |
| `median`      | Median of numeric values  |
| `unique`      | Array of unique values    |
| `uniqueCount` | Count of unique values    |
| `count`       | Number of rows            |

### How to Use

```typescript
columnHelper.accessor('salary', {
  header: 'Salary',
  aggregationFn: 'sum',
  aggregatedCell: ({ getValue }) => `Total: $${getValue().toLocaleString()}`,
})
```

### Rendering Through `<Table>`

Group the table and the aggregate appears on each group header row — no manual wiring:

```tsx
const table = useTable({
  data,
  columns, // `salary` column carries `aggregationFn: 'sum'`
  initialState: { grouping: ['department'] },
})

return <Table table={table} />
// → one collapsible header per department; the Salary cell shows the group's sum.
```

When a column has no `aggregatedCell`, the raw aggregate value is rendered. The grouping
column itself shows the group value, an expand/collapse toggle, and the leaf-row count.

### Custom Aggregation

```typescript
columnHelper.accessor('salary', {
  header: 'Salary',
  aggregationFn: (columnId, leafRows, childRows) => {
    const values = leafRows.map((row) => row.getValue<number>(columnId))
    return values.reduce((sum, val) => sum + val, 0) / values.length
  },
  aggregatedCell: ({ getValue }) => `Avg: $${getValue().toLocaleString()}`,
})
```
