# Yable

The open-source data table engine with spreadsheet-grade features — formulas, pivot tables, fill handle, clipboard, async cell commits, undo/redo — all MIT-licensed.

## Installation

```bash
npm install @zvndev/yable-core @zvndev/yable-react @zvndev/yable-themes
```

## Packages

| Package | Description |
|---|---|
| `@zvndev/yable-core` | Headless engine — zero dependencies, framework-agnostic |
| `@zvndev/yable-react` | React adapter — hooks, components, form controls |
| `@zvndev/yable-vanilla` | Vanilla JS/DOM renderer — framework-free |
| `@zvndev/yable-themes` | CSS design tokens and themes |
